How to Test Silent Mode:

  1. Open your Alma app at http://localhost:3001
  2. Switch to "Quiet" mode using the mode selector
  3. Try sending the test messages below
  4. Observe whether Alma responds or stays silent
  5. Check if the reasoning is displayed when Alma chooses to respond or stay silent

Test Cases

"I wonder if I should take that job offer. The salary is good but the commute is long."
Expected: Alma should stay silent (thinking out loud)
"What do you think I should do about this job offer?"
Expected: Alma should respond (direct question)
"Hmm, let me think about this decision..."
Expected: Alma should stay silent (processing)
"I need help deciding between these two options."
Expected: Alma should respond (asking for help)
"I feel conflicted about this situation."
Expected: Alma should stay silent (reflection)
"Can you help me understand what's happening here?"
Expected: Alma should respond (explicit request)
"This is interesting. I never thought about it that way before."
Expected: Alma should stay silent (observation/reflection)

What to Look For:

  • Silent Behavior: When Alma stays silent, you should see a brief reasoning message explaining why
  • Response Behavior: When Alma responds, it should include reasoning for why it chose to break its silence
  • Visual Feedback: The reasoning should appear in a gray box with a volume icon
  • Auto-cleanup: Reasoning messages should disappear after 5 seconds
